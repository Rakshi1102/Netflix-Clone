Netflix Clone 🎬
A simple and responsive Netflix Clone built using HTML and CSS. This project recreates the look and feel of the Netflix landing page with a modern UI, trending movies section, FAQ section, and responsive layout design.
🚀 Features


Responsive Netflix-inspired homepage


Hero section with background banner


Trending movies/shows section


FAQ section


Email signup UI


Footer with navigation links


Clean and modern design


🛠️ Technologies Used


HTML5


CSS3
